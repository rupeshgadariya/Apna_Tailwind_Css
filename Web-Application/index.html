01 · Setup

Installation

Add the CDN script tag before the closing </body> tag of your HTML file. No npm, no configuration needed.

HTML 
<!-- Add before </body> -->
<script src="https://cdn.jsdelivr.net/gh/rupeshgadariya/Apna_Tailwind_Css@main/apnaCss.js"></script>

Then add the chai class to any element you want to style, along with your utility classes:

HTML 
<h1 class="chai text-red m-10">Hello World</h1>

<div class="chai flex justify-center items-center p-20 bg-lightblue">
    <p class="chai text-lg text-center font-bold">Centered Content</p>
</div>
💡

The chai class is required on every element you want Apna CSS to process. Think of it as the activation class that tells the script which elements to style.

02 · Core

How It Works

Apna CSS is a pure JavaScript implementation. When the script loads, it runs through these steps:

Step 1 Selects all elements with the chai class
Step 2 Parses all utility class names on each element
Step 3 Matches each class name to a CSS property
Step 4 Applies styles and combines complex transforms
03 · Utilities

Colors

Text Color

Use text-<color> to set font color. You can use named colors or hex values.

text-red text-blue text-green text-darkblue text-gray text-#ff0000
HTML
<p class="chai text-red">Red text</p>
<p class="chai text-blue">Blue text</p>
<p class="chai text-#28a745">Custom hex color</p>

Background Color

Use bg-<color> to set background color.

bg-red bg-blue bg-lightgray bg-lightblue bg-#000000

Positioning

absolute relative fixed sticky

Use directional classes to set position offset values:

top-{n}top: {n}px

14 · Reference

Full Example

A complete page layout using only Apna CSS utility classes:

HTML
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Apna CSS Demo</title>
</head>
<body>

  <!-- Hero Section -->
  <div class="chai flex justify-center items-center p-20 bg-lightblue">
    <h1 class="chai text-2xl text-center text-darkblue font-bold m-10">
      Welcome to Apna CSS
    </h1>
  </div>

  <!-- Card -->
  <div class="chai p-20 m-10 border-2 border-color-gray rounded-10 shadow-md">
    <p class="chai text-lg text-gray">
      A lightweight CSS framework.
    </p>
  </div>

  <!-- Button -->
  <button class="chai p-10 m-5 bg-blue text-white rounded-5 cursor-pointer transition">
    Click Me
  </button>

  <script src="https://cdn.jsdelivr.net/gh/rupeshgadariya/Apna_Tailwind_Css@main/apnaCss.js"></script>
</body>
</html>
